Create competition

When you want to register a new competition, please follow these steps:

  1. You start on the competitions overview page (see above).
  2. On the top-right of the list, you see a green arrow with the text: 'New'. Click on it.
  3. On the page that follows, fill in the competition details.
  4. Click save.
  5. That's it!
