Tournaments functionalities

Last updated: Dec 13, 2020

Important

You must be logged in to access these functionalities!

Tournaments Overview

In the navigation tab (top left of your screen), you click 'Tournaments' to go to the overview page.

Tournament detail page

Follow these steps to go to a tournament's detail page.

  • Start on the tournaments overview page.
  • At the end of the row of the tournament you want to inspect, click this icon: more_vert
  • Click on tournament page.

You now see the tournament's detail page.

Create tournament

When you want to register a new tournament, please follow these steps:

  1. On the navigation tab, click 'Manage items' and select tournaments.
  2. On the top-right of the list, you see a green arrow with the text: 'New'. Click on it.
  3. On the page that follows, fill in the tournament details.
  4. Click save.
  5. That's it!

Important

You must be an admin to access these functionalities!

Delete tournament

When you want to delete a tournament, please follow these steps:

  1. On the navigation tab, click 'Manage items' and select tournaments.
  2. On the row of the tournament you want to delete, click: more_vert
  3. Click 'delete tournament'.
  4. Confirm.
  5. You have now deleted a tournament.

Important

You must be an admin to access these functionalities!

Edit tournament

When you want to edit a tournament's information, please follow these steps:

  1. On the navigation tab, click 'Manage items' and select tournaments.
  2. On the row of the tournament you want to edit, click: more_vert
  3. Click 'edit tournament'.
  4. Confirm.
  5. You have now edited a tournament.

Important

You must be an admin to access these functionalities!

Start tournament

When you want to start a tournament, please follow these steps:

  1. On the navigation tab, click 'Manage items' and select tournaments.
  2. On the row of the tournament you want to start, click: more_vert
  3. Click 'start tournament'.
  4. You have now started a tournament.

Important

You must be an admin to access these functionalities!

Tournament matches

Tournament matches are a bit different than single matches.
Within a tournament, only a moderator can start matches, edit their scores and stop them.
The steps of these features remain the same, but only the moderator can execute them.
