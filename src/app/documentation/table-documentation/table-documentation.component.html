Table functionalities

Last updated: Dec 10, 2020

Important

You must be logged in as an admin to access table functionalities!

Tables overview

To get an overview of all the tables, click on 'Manage items' in the navigation tab, and click on 'Tables' (or click here).
The navigation tab is located top-left of your screen.
This will show you each table, with its name, which company owns it, where it's are located
and who is responsible for that table.

Edit table

Editing a table is easy if you follow these steps:

  1. At the end of a row, you will see this icon: more_vert. Click on it.
  2. A pop-up will show with the following text: 'Edit table'. Click on it.
  3. You are now on the selected table's edit page!
  4. Edit the fields as you like, and click save.
  5. You're done!

Create table

When you want to register a new table, please follow these steps:

  1. You start on the tables overview page (see above).
  2. On the top-right of the list, you see a green arrow with the text: 'New'. Click on it.
  3. On the page that follows, fill in the table details.
  4. Click save.
  5. That's it!

Delete table

To purge a table from the system, follow this path:

  1. You start on the tables overview page (see above)
  2. At the end of a row, you will see this icon: more_vert. Click on it.
  3. A pop-up will show with the following text: 'Delete table'. Click on it.
  4. The application will ask you if you are absolutely sure you want to delete the selected table.
    • If you do not want to delete the selected table, click the blue cancel button.
    • If you do want to delete the selected table, click the red delete button.
  5. You have reached your destination!
