Last updated: Dec 12, 2020

User rankings

Each user's individual wins and losses are tracked.
To give everyone an overview which players are the best (or worst),
we made a ranking mechanism.
Each user has points, which are calculated base on their wins and losses, using this formula:
score = ((total wins/total matches) * 100) * total wins
You can view user rankings by clicking on the link in the navigation tab (or click here).
