User functionalities

Last updated: Dec 10, 2020

Sign Up

If you want to use this application, you'll need an account. To make an account, you can use the sign up functionality
To access this functionality, click on the sign up button on the home page (or click here).
Fill in the necessary fields and confirm. Now you are ready to use the application!

Log In

If you want to use this application, you'll need an account. If you already have an account, you can use the log in functionality.
To access this functionality, click on the login button on the home page (or click here).
Fill in the necessary fields and confirm. Now you are ready to use the application!

User profile

Each user on this application has a profile page. On this profile page, each user can see detailed information.
Examples are: matches played, matches won, the teams a user is part of and much more.
To go to this page, you have to be logged in. If so, click on you name located top-right on your screen.
In the drop down menu, click on My profile.

Important

You must be logged in to access this functionality!
