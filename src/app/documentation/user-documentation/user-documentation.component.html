User functionalities

Last updated: Dec 13, 2020

Sign Up

If you want to use this application, you'll need an account. To make an account, you can use the sign up functionality
To access this functionality, click on the sign up button on the home page (or click here).
Fill in the necessary fields and confirm. Now you are ready to use the application!

Log In

If you want to use this application, you'll need an account. If you already have an account, you can use the log in functionality.
To access this functionality, click on the login button on the home page (or click here).
Fill in the necessary fields and confirm. Now you are ready to use the application!

User profile

Each user on this application has a profile page. On this profile page you can find detailed information about a user.
Examples are: matches played, matches won, the teams a user is part of and much more.
To go to this page, there are several ways:

Your own profile page.

  1. Click on your name in the navigation tab (top right of your screen).
  2. Click on 'My profile'.
  3. There you go!

An other user's profile page.

Again, there are several ways to get to an other user's profile page.

  1. Via a team's detail page: click the username.
  2. Via rankings: click more_vert and click 'Profile page'.

Important

You must be logged in to access this functionality!

User profile picture

Each user on this application can change their profile picture. To do this, follow these steps.

  1. Start on your profile page (see above).
  2. Move your cursor to the profile picture.
  3. The picture will change to a green square with a white pencil inside it. Click it.
  4. A modal will open, with a green button saying 'Choose file'. Click it.
  5. Now you can choose a picture on your computer with windows explorer.
  6. Select the file, and click open (or double-click on the file).
  7. Click save.
  8. That's it! Not too shabby eh?

You can always cancel by clicking the cancel button.

Important

  • You must be logged in to use this feature!
  • You can only do this with your own profile!
